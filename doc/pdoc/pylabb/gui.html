pylabb.gui

pylabb.gui – PyQt6-basierte grafische Benutzeroberfläche

Dieses Paket enthält das vollständige PyQt6-GUI-Frontend für PyLab:

Hauptfenster

MainWindow – Zentrales QMainWindow mit Menüleiste, Werkzeugleiste, Dock-Widgets und Plot-Tabs.

Dock-Widgets (in gui.widgets)

  • ~pylabb.gui.widgets.system_editor.SystemEditorPanel – Strecke G(s) und PID-Regler C(s) bearbeiten
  • ~pylabb.gui.widgets.analysis_widget.AnalysisWidget – Stabilitätsanalyse und Kenngrößen
  • ~pylabb.gui.widgets.verification_widget.VerificationWidget – Subgraph-Äquivalenzprüfung zweier Regelkreise
  • ~pylabb.gui.widgets.bio_classify_widget.BioClassifyWidget – Biologische Äquivalenzklassen-Klassifikation
  • ~pylabb.gui.widgets.codegen_widget.CodegenWidget – MicroPython-Codegenerierung

Plot-Panel (in gui.widgets)

  • ~pylabb.gui.widgets.plot_widget.MultiPlotWidget – Tabbed Plots (Bode, Nyquist, Sprungantwort, Pol-Nullstellen, …)

309    @pyqtSlot()
310    def _on_plot_bode(self) -> None:
311        if not self._check_plant():
312            return
313        try:
314            pw = self._plots.get_plot("Bode")
315            pw.clear()
316            from pylabb.visualization.bode import plot_bode
317            G_list = [self._plant]
318            if self._controller:
319                C_tf = self._controller.transfer_function()
320                from pylabb.control.design import closed_loop, series
321                L = series(C_tf, self._plant)
322                L.name = "L=C·G"
323                G_list.append(L)
324            plot_bode(*G_list, fig=pw.fig, title="Bode-Diagramm")
325            pw.redraw()
326            self._status("Bode-Diagramm aktualisiert")
327        except Exception as ex:
328            self._error("Bode-Fehler", str(ex))
329

359    @pyqtSlot()
360    def _on_plot_step(self) -> None:
361        if not self._check_plant():
362            return
363        try:
364            pw = self._plots.get_plot("Sprungantwort")
365            pw.clear()
366            from pylabb.visualization.time_plots import plot_step_response
367            systems = [self._plant]
368            if self._controller:
369                from pylabb.control.design import closed_loop
370                C_tf = self._controller.transfer_function()
371                T = closed_loop(self._plant, C_tf)
372                T.name = "T(s) ges. Kreis"
373                systems.append(T)
374            plot_step_response(*systems, fig=pw.fig, title="Sprungantwort")
375            pw.redraw()
376            self._status("Sprungantwort aktualisiert")
377        except Exception as ex:
378            self._error("Sprungantwort-Fehler", str(ex))
379
380    @pyqtSlot()
381    def _on_simulate_closed_loop(self) -> None:
382        if not self._check_plant():
383            return
384        if self._controller is None:
385            self._warn("Kein Regler definiert.", "Bitte zuerst einen PID-Regler eingeben.")
386            return
387        try:
388            from pylabb.core.signals import step_signal
389            from pylabb.simulation.time_domain import ClosedLoopSimulator
390            from pylabb.visualization.time_plots import plot_simulation
391
392            C_tf = self._controller.transfer_function()
393            sim = ClosedLoopSimulator(self._plant, C_tf, name="Regelkreis")
394            w = step_signal(t_end=20.0, dt=0.01)
395            result = sim.run(w)
396
397            pw = self._plots.get_plot("Simulation")
398            pw.clear()
399            plot_simulation(result, fig=pw.fig, title="Regelkreis-Simulation")
400            pw.redraw()
401            s = result.summary()
402            self._status(
403                f"Simulation: ts={s['Einschwingzeit [s]']:.3g}s, "
404                f"Ü={s['Überschwingen [%]']:.2g}%, "
405                f"IAE={s['IAE']:.3g}"
406            )
407        except Exception as ex:
408            self._error("Simulationsfehler", str(ex))
